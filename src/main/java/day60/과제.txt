[ 웹 과제6 : 3개월 최종과제 ]
    1. 주제 : 요식업 웨이팅(대기 명단) 서비스 구축
    2. 개발 환경 요구사항
        1] 인텔리제이에서 새로운 gradle 프로젝트 생성하여 진행
        2] Sprint Web MVC2_tier3 패턴 사용
        3] HTML은 Thymeleaf 템플릿 사용
    3. 서비스 요구사항
        1. 대기명단 등록 페이지와 대기명단 목록 페이지로 이동할 수 있게 헤더에 각 페이지 이동버튼 생성
        2. ['대기명단 등록'] 웹에서 고객들에게 전화번호와 인원 수를 입력받아 대기번호를 주는 서비스
        3. ['대기명단 목록'] 웹에서 관리자가 모든 대기 명단 목록을 볼 수 있도록 만들어주세요.

    * 추가 요구사항 [시간이 남으면]
    1. ['대기명단 등록'] 웹에서 현재 대기 명단 수를 출력하시오.
    2. ['대기명단 목록'] 웹에서 모든 대기 명단 목록에서 특정한 대기명단 개별 삭제를 할 수 있도록 삭제버튼
    3. ['마이페이지'] 마이페이지 추가하여 내 전화번호를 입력하면 현재 예상시간을 알려주세요.
        -> 만약에 대기 명단에 없는 전화번호이면 '존재하지 않는 번호입니다'를 출력
        -> 예상시간은 대기명단 1개마다 5분씩 부여하여 만일 3개의 대기명단이 존재하면 [대기 예상시간 : 15분]
